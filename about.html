Some people code to live, but I live to code. Since graduating with a 1st class honours degree in Computer Engineering from Imperial College (world ranking), London, I've worked at compaines of all sizes, from a small four person startup (clipboard.com) [acquired by salesforce.com], to companies with tens of thousands of employees like Microsoft, where I got to compile the windows source code and kernel debug (there is nothing like the first time you step from user mode to kernel mode in a remote debugger).

I've used numerous languages, C, Objective-C, C#, JavaScript, but I mainly code JavaScript at the moment (obviously using the one true editor, emacs), both in the client and the backend via node.js. For more information, see my projects page.

When I'm not hacking on some code, I can be found snapping pictures around the Cascade Mountains in Washington state, or in a sweaty heap at my local crossfit gym.
